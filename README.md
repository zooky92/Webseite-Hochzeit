# Hochzeitseinladung

Diese Website ist als leichte, günstige und statische Hochzeitseinladung gedacht. Sie besteht aus reinem HTML, CSS und JavaScript und kann für wenige Monate auf einfachem Hosting betrieben werden.

## Inhalte
- Überschrift: „Marisa & Denny“
- Einladungstext und Location-Abschnitt
- Rückmeldeformular mit dynamischen Gästefeldern

## Hinweise
- Datum und Adresse in index.html ersetzen.
- Das Formular speichert keine Daten automatisch. Für echte Antworten wird ein Backend oder ein externer Dienst benötigt.
